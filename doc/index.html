Introduzione

Lo scopo del progetto è realizzare un'implementazione dei Grafi di Pianificazione per studiarne le caratteristiche e le euristiche generate. Il progetto è stato sviluppato all'interno dell'università degli Studi di Padova come progetto per il corso di Intelligenza Artificiale.

Utilizzo del programma

Il programma è stato implementato nel linguaggio Java e sviluppato sul operativo macOS.
